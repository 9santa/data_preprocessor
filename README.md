# DataPreprocessor

Небольшой класс на Python для базовой очистки и трансформации табличных данных (pandas DataFrame).

## Возможности
- `remove_missing(threshold=0.5)` — удаляет столбцы, где доля пропусков выше `threshold`, а оставшиеся пропуски заполняет:
  - числовые: `mean` или `median`
  - категориальные: `mode`
- `encode_categorical()` — выполняет one-hot encoding для строковых/категориальных признаков.
- `normalize_numeric(method='minmax' | 'std')` — нормализует числовые признаки (Min-Max или стандартизация).
- `fit_transform(...)` — последовательно применяет все шаги и возвращает обработанный DataFrame.

Дополнительно класс сохраняет информацию о выполненных преобразованиях (удалённые столбцы, параметры заполнения и нормализации, список one-hot колонок).

## Содержимое репозитория
